## 📦 项目

### 🚀 景伴游 · AI 数字人交互导览系统

[![jingbanyou-ai-tour-guide](https://img.shields.io/badge/GitHub-Repository-181717?style=flat-square&logo=github)](https://github.com/chen-foregone/jingbanyou-ai-tour-guide)

> 基于 Spring AI Graph 状态机 + RAG 双路检索的智能景区导游系统

- **架构设计**: Spring AI Graph 状态机实现多意图路由
- **智能问答**: RAG 双路检索 + 数字人对话
- **异步解耦**: RabbitMQ 消息队列
- **流式推送**: SSE 实时流式响应，首字延迟 < 1 秒
- **性能**: 支撑日均万级请求

**技术栈**: Spring Boot &middot; Spring AI Graph &middot; Spring Cloud Alibaba &middot; RAG &middot; RabbitMQ &middot; SSE &middot; MySQL &middot; Redis
