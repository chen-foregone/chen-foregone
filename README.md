## 📦 项目

### 🚀 景伴游 · AI 数字人交互导览系统

[![Repo](https://img.shields.io/badge/Repository-181717?style=flat-square&logo=github)](https://github.com/chen-foregone/jingbanyou-ai-tour-guide)

> 基于 Spring AI Graph 状态机 + RAG 双路检索的智能景区导游系统

- **多意图路由**: Spring AI Graph 状态机，智能分发对话意图
- **双路检索**: RAG 向量检索 + 关键词检索，精准匹配景点知识
- **异步架构**: RabbitMQ 解耦消息处理，支撑日均万级请求
- **流式响应**: SSE 实时推送，首字延迟 < 1 秒
